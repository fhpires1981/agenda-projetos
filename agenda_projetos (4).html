Siga o passo a passo para criar o Apps Script: Passo a Passo → Configurar Google Sheets abaixo. Depois cole o URL gerado no campo abaixo.
Cole aqui o URL gerado ao publicar o Web App no Google Apps Script
Como configurar o Google Sheets (faça uma vez só)
1. Acesse sheets.google.com → crie uma planilha nova → dê o nome agenda_projetos_db
2. No menu superior clique em Extensões → Apps Script
3. Apague o código existente e cole TODO o código abaixo:
4. Clique em Salvar (ícone de disquete)
5. Clique em Implantar → Nova implantação
6. Em "Tipo": escolha App da Web
7. Em "Executar como": Eu (seu email)
8. Em "Quem tem acesso": Qualquer pessoa
9. Clique em Implantar → autorize as permissões → copie o URL da implantação
10. Cole o URL no campo acima e clique em Salvar e Conectar ✅
